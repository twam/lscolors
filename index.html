LS_COLORS Generator

Select your desired colors and copy the corresponding export line to your .profile file! You can also edit the LSCOLORS and LS_COLORS field to convert one string to another. Notice that only values which are understood by LSCOLORS and LS_COLORS are supported!

If you're using zsh as your shell and working on Linux and OS X machines you can get an idea how to add both from my ls_colors.zsh.

Based on an idea of Geoff Greer.

Color selection

File type Bold Foreground Background

BSD/OS X (LSCOLORS)

export LSCOLORS="";

Linux (LS_COLORS)

export LS_COLORS="";
